# 简历项目描述（可直接使用）

## 标准版

设计并落地对话式电商运营 Agent，统一承接库存查询、市场查询、双平台利润筛选与报表导出等业务流程。  
构建重型 RAG 检索链路（Embedding + VectorDB + Rerank + Fallback），提升自然语言场景下 SKU 召回准确率与稳定性。  
引入高风险动作 Dry-Run 机制，实现上架/改价流程的“可演示、可校验、零副作用”编排输出。

## 精简版（面试自我介绍可用）

我做过一个电商运营 Agent 项目，把查询、比价、导出和运营流程编排统一到对话入口。  
在检索层用了 Embedding + 向量库 + Rerank，并做了 TF-IDF 回退，兼顾准确率和稳定性。  
对高风险动作采用 Dry-Run 设计，保证业务安全可控。
