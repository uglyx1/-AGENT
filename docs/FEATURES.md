# 功能模块

## 1. 对话式查询 Agent

- 支持自然语言输入
- 支持库存查询与市场查询场景
- 当商品信息不完整时，可通过检索补全候选结果

## 2. 重型 RAG 检索

- 基于 Embedding 的语义检索，适配口语化与非标准表达
- 向量数据库召回候选
- Rerank 精排提升命中准确度
- 具备 TF-IDF 回退机制，保证低依赖可用性

## 3. 双平台利润对比

- 以当前平台价格为基准，对比第二平台价格数据
- 支持利润阈值筛选（例如：利润 > 100 / 200）
- 返回表格预览与可下载报表元数据

## 4. 报表导出与下载

- 结构化结果行输出
- CSV 导出链路
- 下载接口可直接供前端或业务面板接入

## 5. 运营动作草稿模式（安全优先）

- 支持对话式上架/改价流程草稿生成
- 支持参数完整性校验
- 明确返回 Dry-Run 标记（`executed=false`）
- 公开演示范围内不执行真实写操作
